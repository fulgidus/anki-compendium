## Purpose

The Git Workflow Skill provides comprehensive expertise in version control workflows and collaboration patterns. It enables agents to define branching strategies, enforce Git standards, coordinate code integration, and maintain repository integrity.

---

## When to Load This Skill

Load the Git Workflow Skill when:
- Defining or documenting branching strategies
- Establishing Git workflows for teams
- Creating branch protection rules
- Defining commit message standards
- Resolving merge conflicts
- Setting up code review processes
- Coordinating release branches
- Implementing version control governance
