## Purpose

The Marp Skill enables agents to create professional, consistent, and version-controlled presentation slides using Markdown syntax. Marp transforms plain Markdown into beautiful slides with customizable themes, layouts, and export options.

---

## When to Load This Skill

Load the Marp Skill when:
- Creating presentation decks for pitches, campaigns, or reports
- Building technical documentation with visual slides
- Designing product roadmap presentations
- Creating training materials and tutorials
- Generating stakeholder reports and updates
- Converting Markdown documentation into slide format
- Building reusable presentation templates
